SongWriter is a Free and Open Source tool for creating and editing simple musical lead-sheets. SongWriter uses JSON Song Notation as it's native de-facto data format, and is both a proof of concept as well as a reference implementation for the JSON Song Notation format. One of the goals of SongWriter is to promote and encourage the use the JSON Song Notation format as a modern, compute oriented data format for storing song data.
