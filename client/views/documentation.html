SongWriter Documentation

The SongWriter application is meant to be an aid in the writing and editing of a simple music lead sheet. As such you might compare it (at least structuarly) to a simple word processor dedicated to one particular type of document, i.e the music lead sheet.

Overview

The application display consits of three sections, the toolbar, the console and the editor. The console and the editor are not always visible but are displayed based on user action; this action is expressed by use of the toolbar which is always visible at the top of the application display.

Toolbar

The toolbar provides menu selections that enable interaction with the application including the File, View, Tools and Help top-level menu items. To learn more about using the application toolbar please read the "Toolbar Usage" section..

Console

The console provides an input form that defnes a particualr song's header as well as a chord selection display.
To best understand the SongWriter application we should start with the application toolbar (shown above), which is always present at the top of the application and presents the File, View, Tools and Help top level menu options. If an item is greyed out it means it is currently disabled, for example the Tools menu option shown above.
The File menu is used to start a new lead-sheet or to open an existing lead-sheet for editing:
  • New - Start a new lead sheet.
  • Open - Open an existing leadsheet to edit.
  • Open Recent - Previously opened lead sheets are displayed here. If no files have been opened this option is greyed out.
  • Save - Save the current file being edited on the server. This option is greyed unless there is something to save.
  • Export - Export the lead sheet for download in various formats. This option is greyed as it is currently not implemented.
  • Print - Print the current lead-sheet.
The View menu contains the following options:
  • Hide/Show console - This option is greyed out until lead-sheet is active.
  • Display Format - Greyed out as not yet implemented.
  • Auto Layout - Layout song grid as best fits the screen of the device in use.
  • 4 Columns - Display lead-sheet in a four column grid.
  • 5 Columns - Display lead-sheet in a five column grid.
  • 6 Columns - Display lead-sheet in a six column grid.
  • Print - Print the current lead-sheet.
The Tools menu will contain a metronome among other possible tools. Currently greyed out as not yet implemented.
  • Metronome - Greyed out as not yet implemented.
The Help menu contains the About and Documentation options:
  • About - General Application information.
  • License - Application License information.
  • Documentation - Application documentation.
